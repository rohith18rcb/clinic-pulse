Quick Actions

⚙️ Connect WhatsApp (Twilio)

1
Sign up for Twilio (free)
Go to twilio.com → Create free account → Get $15 trial credit
2
Activate WhatsApp Sandbox
Console → Messaging → Try it out → Send a WhatsApp message
Send join <your-keyword> from your WhatsApp to +14155238886
3
Create your .env file
Copy .env.example → .env
Fill in TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, and your WhatsApp number
4
Install ngrok for webhooks
Download from ngrok.com → Run: ngrok http 3000
Copy the HTTPS URL → Set as Webhook in Twilio Sandbox settings
5
Restart the server
Run npm start — Real WhatsApp messages will now be sent! 🎉

💡 For Demo Purposes

Even without Twilio, the dashboard fully simulates all message flows. Great for showing to clinics before you set up the live integration!
